# Buffer-clone

## About

Buffer is a software application for the web and mobile, designed to manage accounts in social networks, by providing the means for a user to schedule posts to Social Media Pages.






## It is an group Project.


## Features

- Signup
- Login ( user login )
- Navbar
- Home Page
